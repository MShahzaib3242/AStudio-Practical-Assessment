# AStudio Practical Assessment - React-Vite + TS + Tailwind

clone the project with the following command:

```
git clone https://github.com/MShahzaib3242/AStudio-Practical-Assessment.git
cd AStudio-Practical-Assessment
npm install
npm run dev
```
